# @vaultledger/cli

The CLI for setting up and governing an AI agent's memory over an Obsidian vault: initialize the vault, review and approve proposed changes, and roll back with `ledger undo`.

```
npx @vaultledger/cli setup /path/to/vault
```

See the [repo README](https://github.com/Ktopper/VaultLedger#readme) and [docs/GETTING_STARTED.md](https://github.com/Ktopper/VaultLedger/blob/main/docs/GETTING_STARTED.md) for the full command reference.
